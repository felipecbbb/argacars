# ARGA Premium Cars

Sitio web de [ARGA Premium Cars](https://argapremiumcars.es) — importación personalizada de vehículos premium desde Alemania.

## Stack

HTML5 + CSS3 + JavaScript vanilla. Cero dependencias en runtime.

- **Tipografía:** Inter (Google Fonts)
- **Animaciones:** IntersectionObserver + spring easings (Framer-Motion-like)
- **Diseño:** light-first con toggle dark/light persistente
- **Mobile:** drawer fullscreen, footer reorganizado, sliders con touch
